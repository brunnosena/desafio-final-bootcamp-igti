- 💹 [**Node.js**][node]

- 📄 **MongoDB**

  

## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/brunnosena/desafio-final-bootcamp-igti.git
```

#### Backend

```bash
# Go into the repository
$ cd backend

# Install dependencies
$ yarn install

# Run the app
$ yarn start
```

#### Frontend

```bash
# Go into the repository
$ cd frontend

# Install dependencies
$ yarn install

# Run the app
$ yarn start
```
